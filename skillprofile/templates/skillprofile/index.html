Myskills


Website where everyone can track their skill progress.


Used tools:

Django

  • Models links: OneToOne (User<->UserInfo), OneToMany (User<->Skill), ManyToMany (Skill<->Tag)
  • ORM (for several views)
  • CBV, generics, forms
  • Permissions: UserPassesTestMixin, LoginRequiredMixin
  • Tests
  • Debug-tool-bar

Boostrap

  • JS and CSS are added to static and available to all apps.
  • Base template has navbar and footer.
  • Cards with used tools on the main page.

Docker

Project is dockerized, application and database are deployed on the same network using docker-compose.

Postgres

Postgres is project database.
